Task Time Tracker

Backlog・GitHubの作業時間を自動計測するChrome拡張機能

🔄 自動時間計測

タスクのステータス変更を検知して自動で作業時間を計測。手動記録は不要です。

📊 詳細レポート

日次・週次の作業時間をタスク別に集計。生産性の分析と改善に活用できます。

🌐 複数サービス対応

Backlog・GitHub Projectsの両方に対応。一つの拡張機能で統合管理。

🔒 プライバシー保護

すべてのデータはブラウザ内のみに保存。外部送信は一切行いません。

対応サービス

  • Backlog (*.backlog.jp / *.backlog.com)
    • カンバンボードでのタスク移動検知
    • 課題ステータス変更の自動追跡
  • GitHub (github.com)
    • GitHub Projectsでのカード移動検知
    • Issue/PRステータス変更の追跡

主な機能

  • ステータス変更による自動計測開始・終了
  • リアルタイム作業時間表示
  • 日次・週次レポート生成
  • タスク別作業時間の詳細分析
  • カスタマイズ可能な計測設定
